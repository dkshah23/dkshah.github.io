FORM VALIDATION

First Name

  • Name cannot contain Number nor Symbol


Last Name

  • Name cannot contain Number nor Symbol


Phone Number

  • Number must starts with 6-9 and must be of 10 digit.


Present Address





Temporary Address

  • If Temporary Address is same as Present Address select checkbox.


E-mail Address

  • E-mail Address should not end with @, _ or Number.


Password

  • Password must contain UPPERCASE(A~Z).

  • Password must contain LOWERCASE(a~z).

  • Password must contain NUMBER(0~9).

  • Password must contain SYMBOLS(@,?,#,$).

  • Password length must be 6 to 12 in length.

Re-Password

  • Re-Password must be same as original Password.
